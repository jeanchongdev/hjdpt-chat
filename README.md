
### Installation

- Clone the repository
  ```sh
  git clone https://github.com/jeanchongdev/hjdpt-chat.git
  ```
- Navigate to repository folder
  ```sh
  cd hjdpt-chat
  ```
- Install npm packages
  ```sh
  npm install
  ```
- To run the app in the development mode run in the project directory: 
  ```sh
  npm start
  ```
